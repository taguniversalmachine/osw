MidiOutput [Help Patch]

Output MIDI messages

Syntax

MidiOutput  options

Options

    -driver
    The name of a MIDI device to use for output. MIDI output device names are of the form /midiOut/n
    Type: String

    Inlets

      message
      MIDI message to output
      Type: MidiMessage

    Comments

    You can double click the MidiOutput transform to view and select from a list of available MIDI output devices.

    To produce MIDI messages, use the MidiPack or MidiPackBytes transform. Other transforms listed below also produce MIDI messages

    Known Issues

    MIDI Output is currently available on Windows and Linux platforms.
