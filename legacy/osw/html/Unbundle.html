Unbundle

Unbundle a Bundle into its components.

Syntax

Unbundle options

Options

    -name
    Optional name of transform.
    Type: String

Inlets

    bundleIn
    The Bundle to be unbundled.
    Type: Bundle

Outlets

    out1 ... outn
    Outlets for the unbundled data.
    Type: Any

Comments

Unbundle is used when you want to access data components of a Bundle separately.

If all the outlets of an Unbundle are connected, an additional outlet is automatically added. Thus it is not necessary to know in advance how many connections are required, and more connections can be easily added at any time.
