TimeMachine <

Control virtual time

Syntax

TimeMachine options

Options

    -name
    Optional name for TimeMachine

Inlets

    rate
    Rate of "virtual time" vs. real time.
    Type: Float
    Default: 1
    goto
    Jump to a particular point in " virtual time
    Type: Double
    zero
    Jump to virtual time zero. Same as sending 0 to goto inlet.
    Type: Unit
    timeIn
    Slave to another time machine.
    Type: Time

Outlets

    timeOut
    "Virutal time" output
    Type: Time

Comments

For more information on virtual time control, refer to the chapter on Time Control in OSW.
