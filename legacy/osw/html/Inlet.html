Inlet [Help Patch]

Route a source to a synchronously selected destination.

Syntax

Inlet options

Options

    -name
    Name of the inlet.
    Type: String
    Default: "in"
    -type
    Type specification of the inlet.
    Type: String
    Default: "Float"

Inlets

    None

Outlets

    out
    Emits data received by the inlet.
    Type: depends on the value of -type.

Comments

Inlet is used to create a inlet for patch when it is used as a subpatch or a transform in another patch. Data sent to such an inlet will be received by the corresponding Inlet transform in the subpatch.
