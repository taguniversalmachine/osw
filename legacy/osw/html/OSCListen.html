OSCSend [Help Patch]

Read incoming OSC messages from a specified port.

Syntax

OSCSend options

Arguments

    -name
    Optional name for transform
    -port
    Port number
    Type: Integer
    Default: 7005

Outlets

    messageOut
    Received OSC message
    Type: List

Comments

OSCListen receives OpenSound Control messages on the port specified by the option -port and converts them to OSW lists. The messages returned by OSCListen can be routed to OSW variables via OSCDispatch or to custom processes using OSCRoute.
