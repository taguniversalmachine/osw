Outlet [Help Patch]

Route a source to a synchronously selected destination.

Syntax

Outlet options

Options

    -name
    Name of the outlet.
    Type: String
    Default: "out"
    -type
    Type specification of the outlet.
    Type: String
    Default: "Float"

Inlets

    in
    Data to be emitted by outlet.
    Type: depends on the value of -type.

Outlets

    None

Comments

Outlet is used to create a outlet for patch when it is used as a subpatch or a transform in another patch. Data sent to an Outlet transform of a subpatch will be emitted by the corresponding outlet in the patch that contains it.
