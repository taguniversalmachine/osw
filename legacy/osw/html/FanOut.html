FanOut [Help Patch]

Route a source to several destinations.

Syntax

FanOut options

Options

    -outputs
    Number of initial outlets.
    Type: Integer
    Default: 2

Inlets

    in
    The source data to be routed.
    Type: Changeable

Outlets

    out1 ... outn
    Outlets for the routed data.
    Type: depends on the type connected to in.

Comments

FanOut is required when you want connect the output of one transform to multiple inlets on other transforms:

The -outputs option can used to specify the number of outlets from the FanOut (the default is 2). However, if all the outlets of a FanOut are connected, an additional outlet is automatically added, as in the figure above where both of the original outlets have been connected to other transforms. Thus it is not necessary to know in advance how many connections are required, and more connections can be easily added at any time.
