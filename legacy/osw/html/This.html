This [Help Patch]

Patch Self Referencing

Syntax

This

Options

none

Inlets

Inlets match the inlets of the patch

Outlets

Outlets match the outlets of the patch

Comments

The This transform is used to build recursive self-referencing patches. When This is activated, a clone of the patch is created, and input into the This transform is routed to the corresponding inlets of the cloned patch. Output via outlets of the cloned patch are routed back to the outlets of the This transform.

Known Issues

This does not currently work with some of the math transforms, although it does work with arithmetic operators and Expr.
