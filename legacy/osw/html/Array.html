Array [Help Patch]

Create an array of identical transforms.

Syntax

Array  num  transformName

Inlets

Each inlet matches an inlet of the sub-transform, and should be passed a Bundle of values for them, respectively.
Type: Bundle

Outlets

Each outlet matches an outlet of the sub-transform type, and will output a Bundle of the corresponding data type.
Type: Bundle
