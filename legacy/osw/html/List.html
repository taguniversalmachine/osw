List [Help Patch]

Define and store a list of values.

Syntax

List  elem1   elem2 ...

Inlets

    unitIn
    Output a copy of the defined list whenever a unit is received.
    Type: Unit

Outlets

    listOut
    Connect this outlet to transforms that require the defined list.
    Type: List
