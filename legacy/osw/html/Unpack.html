Unpack [Help Patch]

Unpack several inputs from a single list

Syntax

Unpack options

Options

    -outputs
    Number of outputs from Unpack
    Type: Integer
    Default: 2

    Inlets

      listin
      The list to unpack
      Type: List

    Outlets

      in1 ... inn
      Outlets for the unpacked elements.
      Type: Any.

    Comments

    Unpack is used to separate a list into individual elements, emitted via each of the outlets to the transform. If there are more outlets than elements in the input list, the remaining outputs emit nothing. If the list is larger than the number of outlets, the excess elements of the list are discarded.

    The number of outlets to the Unpack transform automatically grows as connections are added.
