SysexOutput [Help Patch]

Output MIDI system-exclusive messages

Syntax

SysexOutput  options

Options

    -driver
    The name of a MIDI device to use for output. MIDI output device names are of the form /midiOut/n
    Type: String

    Inlets

      message
      MIDI message to output
      Type: MidiMessage

    Comments

    You can double click the SysexOutput transform to view and select from a list of available MIDI output devices.

    Sysem-exclusive messages are streams of bytes beginning with F0 hexidecimal and ending with F7 (also hexidecimal). The actual messages are specific to different MIDI devices such as synthesizers, effects processors or computer soundcards. Consult the manuals for your devices to find the actual messages supported. To enter system-exclusive message streams, create a list (using any of the List transforms or a MessageBox and the List2Raw transform.

    Known Issues

    MIDI Sysex output is currently available on Windows and Linux platforms. There is currently no support for incoming system-exclusive messages.
